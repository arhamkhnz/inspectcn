# Privacy Policy for Inspectcn

Last updated: March 19, 2026

Inspectcn is a Chrome extension that helps users inspect and extract shadcn-style theme tokens from websites.

## Summary

Inspectcn processes theme-related page data locally in the browser so it can:

- read active theme tokens from the current page
- let users copy extracted theme CSS
- save a captured theme for reuse
- apply a saved theme to a localhost preview

Inspectcn does not sell personal data, does not use data for advertising, and does not send extracted theme data to a remote server.

## Data Inspectcn Accesses

When a user opens Inspectcn on a page, the extension may access:

- the current page URL
- the current page title
- CSS custom properties and theme token values available on the page
- limited page styling information needed to detect the active theme

This access is used only to provide the extension's theme inspection features.

## Data Inspectcn Stores

Inspectcn stores data locally in the browser using extension storage. This may include:

- the badge position
- the most recently captured theme snapshot
- the captured page title
- the captured page URL
- localhost preview mappings used to apply a saved theme during development

This data is stored only so the extension can preserve user actions and reuse captured themes later.

## What Inspectcn Does Not Collect

Inspectcn does not intentionally collect or process:

- passwords
- payment information
- health information
- contacts
- private messages
- precise location data

Inspectcn is not designed for advertising, analytics profiling, or background tracking.

## Data Sharing

Inspectcn does not sell, rent, or transfer user data to third parties.

Inspectcn does not send captured theme snapshots, URLs, or page titles to a remote backend operated by the developer.

## Remote Code

Inspectcn does not load or execute remote code. All executable code is packaged with the extension.

## Data Retention and User Control

Users can remove stored extension data by:

- using the extension controls that clear saved theme data
- removing localhost preview state from the extension
- clearing the extension's site data through the browser
- uninstalling the extension

Inspectcn retains locally stored data only until the user removes it.

## Security

Inspectcn is designed to minimize retained data and keep processing local to the browser wherever possible. However, no software can guarantee absolute security.

## Changes to This Policy

This privacy policy may be updated if Inspectcn's functionality changes. The latest version will be published at the same public policy URL provided in the Chrome Web Store listing.
